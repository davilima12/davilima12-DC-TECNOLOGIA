# QuickStart


### Requisitos: 

```
PHP 8.2.0
Composer version 2.4.4
```


### Instalar Dependencias 

```
composer install
```

### Rodar o Projeto:

```
php artisan serve

```

### Configurações:
#### Configurar os dados do seu banco no arquivo .env
#### Depois executar o comando para criar as tabelas e popular sua base de dados
```
php artisan migrate --seed

```
